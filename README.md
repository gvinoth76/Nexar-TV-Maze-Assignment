Assignment of Nexer - TV Maze app

Requirements:

Node - v12.19.0

NPM  - 6.14.8

How to Run:

1. npm install
2. npm start

Application Web url :

By default application will start in http://localhost:3000/ 

To Build:

1. npm run build
2. serve -s build  => to start the build package
